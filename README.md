## Description 💭₊˚
A complete rebuild of my personal portfolio with a **Frutiger Aero / Aqua** aesthetic and **Apple-style** scroll animations. Built from scratch with vanilla HTML, CSS, and JavaScript — no Bootstrap, no framework. Features include glassmorphism, animated aurora backgrounds, floating bubbles, magnetic buttons, 3D card tilt, and smooth scrolling.

## Technologies Used 🎧✧˚.

* HTML5
* CSS3 (custom properties, backdrop-filter, grid, custom keyframes)
* JavaScript (vanilla)
* [GSAP](https://gsap.com/) + ScrollTrigger — scroll-driven reveals & parallax
* [Lenis](https://github.com/darkroomengineering/lenis) — smooth scrolling
* [Bootstrap Icons](https://icons.getbootstrap.com/) — iconography
* Google Fonts — Instrument Serif, Inter, VT323

## Features ✦˚.
* **Frutiger Aero aesthetic** — sky-blue glass, animated aurora blobs, floating bubbles, chrome buttons with shine
* **Apple-style animations** — smooth scroll, scroll-triggered text reveals, magnetic buttons, 3D tilt on cards, mouse parallax
* **Custom cursor** — glowing aqua dot with halo that expands on interactive elements
* **Accessibility** — respects `prefers-reduced-motion`, touch-device fallbacks, semantic HTML
* **Responsive** — mobile-first grid, adapts from 320px to ultrawide

## Website Sections 🎀༘⋆

1. **Hero** — name, status chip, call-to-action with floating glass orb.
2. **About** — background, education, and quick stats.
3. **Skills** — twelve glass cards covering web, mobile, security, and soft skills.
4. **Work** — six selected projects across desktop, mobile, web, and security.
5. **Contact** — sparkle card with email CTA and social links.
